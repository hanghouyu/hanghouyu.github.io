字符串

如果字符串中出现了引号，那么外面就用与其不同的引号括起来
如果里面单引号外面就用双引号，反之亦然
转义符\

操作符

x+y:拼接 n*x或x*n复制n次字符串 x in s： 判断x是不是s的子串

处理函数

len(x) 返回字符串x的长度
str(x) 将任意类型的数据转换成字符串类型
hex(x)和oct(x)将整数x转换为16进制和8进制
chr(U),ord(x)Python使用Unicode编码使用
str,lower(),str.upper()大小写转换
str.split(sep=none),返回列表，由sep决定分隔
str.count(sub)返回sub在str中出现的次数
str.replace(old,new),例如”python”.replace(“n”,”n123.io”)输出为”python123.io”
str.center(width[,fillchar]),居中和填充，”python”.center(20,”=”)结果为’=======python=======’
str.strip(chars)将str中的chars字符全部去掉
str.join(iter)在iter变量除最后元素外每个元素后增加一个,例如str”,”.join(“12345”)输出为”1,2,3,4,5”
