集合类型

多个元素的无序组合，集合中数据类型不可变,元素各不相同
集合用大括号{}表示，元素之间用，分隔
建立集合类型用{}或set()
建立空集合用set()
如果元素重复会直接去掉

操作符

S|T 并 ：返回一个S和T的所有元素
S-T 差： 返回一个在集合S中而不在T中的元素
S&T 交： 返回一个同时在集合S和T中的元素
S^T 补： 返回一个新集合包括S和T中不相同的元素
S<=T 或 S< T 返回True/False, 判断S和T的子集关系
S>=T 或 S> T 返回True/False, 判断S和T的包含关系

增强操作符

&= 换位与赋值运算符 a&=b 等价于 a= a&b
|= 按位或赋值运算符 a|=b 等价于 a=a|b
^= 按位异或赋值运算符 a^=b 等价于 a=a^b

处理方法

S.add(x):如果不在集合S中，将x增加到S
S.discard(x):移除S中元素x,如果x不在集合S中，不报错
S.remove(x):移除S中元素x，如果x不在集合S中，产生KeyError异常
S.clear():移除S中所有元素
S.pop(): 随机返回S的一个元素，更新S，若S为空产生KeyError
S.copy():返回集合S的一个副本
len(S):返回集合S的元素个数
x in S:判断S中元素X，x在S中返回True，否则False
x not in S :与上一个相反
set(x): 将其他类型变量x转换为集合类型
遍历集合可以用for和while

    序列类型

    序列是具有先后关系的一组元素，是一维元素向量，元素类型可以不同
    序列类似数学元素序列
    它有正向递增序号，反向递减序号

    序列处理函数和方法

    x in s 如果x是序列s的元素，返回True，否则返回False
    x not in s 和上面相反
    s + t 链接两个序列s和t
    s*n 或 n*s将序列s复制n次
    s[i] 索引
    s[i:j]或s[i:j:k]切片
    s[::-1]取反
    len(s) 返回序列s的长度
    min(s) 返回序列s中最小元素
    max(s) 返回序列s中的最大元素
    s.index(x) 或 s.index(x,i,j)返回序列s从i开始到j位置中第一次出现元素x的位置
    s.count(x)返回x在s中出现的次数

    元祖类型

    元组一旦创建不能修改，用小括号(),或者tuple()创建,元素用逗号，分隔
    元组类型继承了序列类型的全部通用操作
    元组索引可以套娃：
    color[-1][2]:表示最后一个元素中的第三个元素

    列表类型

    列表类型创建后可以随意修改
    使用[]或者list()创建，元素间用逗号，分隔
    列表中个元素类型可以不同，无长度限制

    操作函数和方法

    ls[i]=x 替换ls第i元素为x
    ls[i:j:k] = lt 用列表lt替换ls切片后所对应元素子列表
    del ls[i] 删除列表ls中第i个元素
    del ls[i:j:k]删除列表ls中第i到第j以k为步长的元素
    ls += lt 更新列表ls
    ls *=lt 更新列表ls
    ls.append(x)在列表ls最后增加一个元素x
    ls.clear()删除列表ls中所有元素
    ls.copy生成一个新列表，赋值ls中所有元素
    ls.insert(i,x)在列表ls中第i位置增加元素x
    ls.pop(i)在列表中取出并删除这个元素
    ls.remove(x)将列表中出现的第一个元素x删除
    ls.reverse()将列表中元素反转

文件的打开文件

r 只读，默认值，如果文件不存在，返回FileNotFoundError
w 覆盖写模式，文件不存在则创建，存在则完全覆盖
x 创建写模式，文件不存在则创建，存在则返回FileExistsError
a 追加写模式，文件不存在则创建，存在则在文件最后追加内容
b 二进制文件模式
t 文本文件模式，默认值
+ 与r/w/x/a一同使用，在原功能基础上增加同时读写功能

文件关闭

.readlines(hint=-1)读入文件所有行，以每行为元素形成列表，如果给出参数，读入前hint行

os库

路径操作

os.path用于操作和处理文件路径
import os.path
import os.patu as op

  1. os.path.abspath(path)返回path文件在当前系统中的绝对路径
  2. os.path.normpath(path)归一为path的表示形式，统一用\ \分隔路径
  3. os.path.relpath(path)返回当前程序与文件之间的相对路径
  4. os.path.dirname(path)返回path中的目录名称
  5. os.path.basename(path)返回path中最后文件的名称
  6. os.path.join(path,*path)组合path与paths,返回一个路径字符串
  7. os.path.exists(path)判断path对应文件或目录是否存在，返回True或者False
  8. os.path.isfile(path)判断path所对应的是否已存在的文件，返回True或者False
  9. os.path.isdir(path)判断path所对应是否为已存在的目录，返回True或者False
  10. os.path.getatime(path)返回path对应文件或目录上一次的访问时间
  11. os.path.getmtime(path)返回path对应文件或目录最近一次的修改时间
  12. os.path.getctime(path)返回path对于文件或目录创建时间
  13. osos.path.getsize(path)返回path对应文件的大小
